Tablas y gráficos

Organiza datos, interpreta frecuencias y transforma la información de una tabla en una representación gráfica clara para comparar categorías.

Panorama del tema

Las tablas permiten ordenar datos y los gráficos ayudan a visualizarlos y compararlos. Cuando los datos están bien organizados, es más fácil identificar cantidades, relaciones y tendencias.

Frecuencia

Indica cuántas veces aparece una categoría o respuesta.

Frecuencia relativa

Muestra qué parte del total representa cada categoría.

Comparación

Los gráficos de barras permiten comparar rápidamente qué grupo tiene mayor o menor cantidad.

Conceptos clave

Tabla de datos

Organiza información en filas y columnas para hacer lecturas ordenadas y comparaciones.

Tabla marginal

Resume los totales por filas o columnas para ver cantidades globales de cada grupo.

Gráfico de barras

Representa cada categoría con una barra cuya altura o longitud depende de la frecuencia.

Ejemplo guiado

Categoría Frecuencia Frecuencia relativa
Fútbol 8 8/20 = 0.40
Baloncesto 5 5/20 = 0.25
Natación 7 7/20 = 0.35
Primero se suma el total. Después se divide cada frecuencia entre ese total para hallar la frecuencia relativa.

Explorador interactivo

Pista: cambia las frecuencias y observa cómo se transforman la tabla, el total y el gráfico de barras.

Cada barra representa una categoría. Cuanto mayor sea la frecuencia, mayor será la barra.

Práctica guiada

Reto 1: cambia una frecuencia y explica cómo se modifica la frecuencia relativa de las demás categorías.
Reto 2: identifica cuál categoría representa menos del 30% del total.
Reto 3: construye un ejemplo en el que dos categorías tengan la misma frecuencia.

Aplicación en contexto

Las tablas y gráficos se usan para organizar encuestas, registrar preferencias, comparar resultados y comunicar información de forma clara. Leer una tabla y pasarla a gráfico ayuda a interpretar mejor los datos.

Ordenar datos permite analizarlos. Graficarlos permite compararlos rápidamente.
