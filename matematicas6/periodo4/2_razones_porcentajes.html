Razones, proporciones y porcentajes

Compara cantidades mediante razones, reconoce proporciones equivalentes y expresa partes de un total usando porcentajes en contextos de encuestas, compras y repartos.

Panorama del tema

Una razón compara dos cantidades. Una proporción afirma que dos razones son equivalentes. Un porcentaje expresa una cantidad como parte de 100.

Razón

Se puede escribir como a:b o a/b. Ejemplo: 3 lápices por cada 5 cuadernos.

Proporción

Si 2/4 = 3/6, entonces las razones forman una proporción.

Porcentaje

25% significa 25 de cada 100, es decir, 25/100 o 0,25.

Conceptos clave

Simplificar razones

Se divide cada cantidad por el mismo número para encontrar una forma equivalente más sencilla.

12:18 = 2:3.

Proporcionalidad

Si una cantidad aumenta o disminuye manteniendo la misma relación, existe proporcionalidad.

1/2 = 2/4 = 4/8.

Porcentaje en contexto

Sirve para describir descuentos, resultados de encuestas o partes coloreadas de un conjunto.

30% de 50 = 15.

Práctica guiada

Reto 1: si 8 de 20 estudiantes prefieren leer, ¿qué porcentaje representan?
Reto 2: decide si 2/3 y 4/6 forman una proporción.
Reto 3: simplifica la razón 18:24 y explica qué significa.

Aplicación en contexto

Las razones y porcentajes se usan para comparar resultados de encuestas, describir descuentos, analizar recetas y calcular avances. Entender estas relaciones prepara el paso hacia la probabilidad y el análisis de datos.

Toda comparación necesita saber con claridad cuál es la parte y cuál es el total.
