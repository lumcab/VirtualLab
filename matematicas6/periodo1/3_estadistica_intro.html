Introducción a la estadística

Descubre cómo recolectar, organizar e interpretar datos a partir de conceptos como población, muestra y variables, para responder preguntas sobre situaciones reales.

Teoría esencial

La estadística estudia datos para describir una situación y tomar decisiones. Antes de hacer tablas o gráficos, es importante saber qué se observa, a quién se estudia y qué característica se quiere analizar.

Población

Es el conjunto total que queremos estudiar. Ejemplo: todos los estudiantes de un grado.

Muestra

Es una parte de la población elegida para observarla con más detalle.

Variable

Es la característica que se registra, como color favorito, edad o número de hermanos.

Interacción

Clasificador de variables

Selecciona una situación, identifica el tipo de variable y observa cómo se organizan sus frecuencias.

Pregunta estadística:

Observa qué se está registrando y decide si la respuesta se expresa con categorías o con números.

CategoríaFrecuencia

Las frecuencias muestran cuántas veces aparece cada respuesta en la muestra observada.

Práctica guiada

Reto 1: si preguntas por el color favorito de 20 estudiantes, ¿la variable es cualitativa o cuantitativa? ¿Por qué?
Reto 2: si registras cuántos libros leyó cada estudiante en un mes, ¿qué tipo de variable estás observando?

Al resolver estos ejemplos, identifica primero la población, luego la muestra si existe, y finalmente la variable que se va a registrar.
